JavaScript中单引号('')和双引号("")的使用只有一个区别，它归结为您必须使用反斜杠(\)字符来转义哪个引号字符:\'或\"。

当使用双引号""创建字符串文字时，双引号字符需要使用反斜杠\"进行转义。

反斜线( ` ` )文字有三大优势:

  1. 更简单的字符串串联(“变量插值”)
    "string "+variable变成了`string ${variable}`
  2. 无需转义(\)单引号或双引号
    "\"Hello World!\""变成`"Hello World"`
  3. 多行代码不带新行字符(\n )
    "Hello\nWorld!"变为`Hello
    World`

使用ESLint保持一致性

如果JavaScript中一致的报价风格对你很重要，就像对Airbnb的工程师一样，那么用 ESLint 很容易解决:

  • ESLint quotes规则可能要求使用双引号(默认)、单引号或反勾号。
  • 引号规则还可以强制一种类型的引号，除非字符串包含引号字符，否则必须对其进行转义。
  • 最后，ESLint可能需要单引号或双引号，但仍然允许字符串使用反斜线文字。
