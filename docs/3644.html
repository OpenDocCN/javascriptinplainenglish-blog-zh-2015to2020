JSON是什么？

JSON 代表JavaScript Object Notation，它是一种存储和传输数据的格式，当数据从服务器发送到网页时经常使用。JSON语法源自JavaScript对象表示法语法，但是JSON格式是纯文本的。生成JSON数据的代码可以用任何编程语言编写，而不仅仅是JavaScript。
