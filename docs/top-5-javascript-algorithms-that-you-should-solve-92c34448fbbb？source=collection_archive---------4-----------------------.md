# 什么是算法？

算法是解决问题或实现特定结果所遵循的一系列步骤。解决一个算法，首先要了解问题，然后用编码来解决。为了使求解算法更容易，你需要将它们分解成小部分。那么单独解决每个部分会比解决整个问题容易很多。算法是非常重要和有用的。他们教你如何解决编码问题，让你成为一名优秀的程序员。
