首先，将上述存储库克隆到本地目录。设置vercel命令行并运行“vercel”命令来部署项目:

使用Vercel设置和部署

$ npm install vercel -g$ cd <project-dir>
$ npm install
$ vercel
  • Vercel将生成一个”。vercel "目录，不要共享或提交此目录。
  • 部署后，打开如下URL:
    —http://YOUR-VERCEL-APP-id . VERCEL . APP—
    示例:https://VERCEL-express-3 . ng duc . VERCEL . APP

app/index.js

  • 如果您使用的是带有节点的typescript，则构建您的源目录(使用类似于“Typescript”包中的“tsc”命令行这样的工具)，输出到“app”。
  • update package . JSON:“main”:“app/index . js”
