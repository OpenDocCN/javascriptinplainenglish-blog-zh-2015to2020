什么是JWT ？

JSON Web Token (JWT)是一个开放标准( RFC 7519 )，它定义了一种紧凑且独立的方式，以JSON对象的形式在各方之间安全地传输信息。该信息可以被验证和信任，因为它是数字签名的。jwt可以使用秘密(使用 HMAC 算法)或使用 RSA 的公钥/私钥对进行签名。
