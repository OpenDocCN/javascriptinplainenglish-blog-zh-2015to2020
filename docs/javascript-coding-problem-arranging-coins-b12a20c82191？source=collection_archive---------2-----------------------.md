# 问题陈述

你总共有 **n 个**硬币，你要组成一个楼梯形状，其中每**个第 k 个**行必须正好有**个 k 个**硬币。

给定 **n** ，求可以形成的**全**楼梯排的总数。

**约束:-** **n** 为非负整数，在 32 位有符号整数的范围内。
