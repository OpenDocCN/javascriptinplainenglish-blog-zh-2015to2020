为数据驱动表单引入碳成分映射器

原文：https://javascript.plainenglish.io/introducing-carbon-component-mapper-for-data-driven-forms-c9221b604d8e?source=collection_archive---------7-----------------------#2020-09-29

数据驱动表单团队刚刚发布了碳成分映射器，它将IBM React碳成分集成到数据驱动表单中。

关于数据驱动表单

数据驱动表单是一个开源 React库，它使用数据驱动方法构建React表单。这种方法基于将JSON模式呈现为React表单，由呈现器提供所有需要的功能。它包括诸如验证、条件字段等许多功能。它帮助web开发人员更快、更简单地编写表单，并在整个应用程序中实现一致性。

关于地图绘制者

映射器是一组与数据驱动表单API集成的组件。这个集合允许用户编写表单，而不需要实现他们自己的组件，因此他们可以立即编写具有广泛功能(显示错误、向导表单等)的完整工作表单。)您还可以检查其他映射器，包括来自 MaterialUI 、 Ant Design 或 BlueprintJS 的组件。

关于碳设计系统

Carbon Design System 是一套用于构建现代web应用的模式、规则和组件。这个系统是完全开源的，由IBM维护。

提供的功能

  • 表单状态管理、验证、条件等等。
  • 摇树 :在CommonJS、ESM或UMD包中选择，以达到最小的包大小。
  • Typescript定义。
  • 附加组件和附加道具:
  • 所有组件都支持helperText，
  • 所有组件都显示错误文本。
  • 向导表单(见下文)。)

提供的组件

文本字段(文本输入)

文本区

收音机

复选框

  • 单个 / 多个变体

日期选择器

时间选择器

  • 包括上午/下午和时区选择器

开关(拨动)

选择 / 多选

  • 允许延迟加载初始数据

子表单

  • 允许将表单分成子组

明文

  • 允许呈现任何文本

滑块

标签页

向导(自定义组件)

  • 分支路径
  • 交互式导航
  • 仅提交访问过的值

双重列表选择(自定义组件)

  • 允许在两个列表之间移动选项
  • 过滤、分类

FieldArray (自定义组件)

  • 允许将表单域动态添加到表单中
  • 即一次注册多个用户

表单模板

  • 表单(间距、按钮)是根据Carbon的指导方针设计的

安装

npm install --save @data-driven-forms/carbon-component-mapper

或者

yarn add @data-driven-forms/carbon-component-mapper

欲了解更多信息，请访问文档页面。
