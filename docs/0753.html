Reactide的设置方式有两种，第一种是捆绑电子app，作为原生桌面App运行。说明如下:

git checkout 3.0-release
npm install
npm run webpack-production
npm run electron-packager
  1. 在您的Reactide文件夹中，导航到release-builds文件夹并双击Reactide(可执行文件)。
