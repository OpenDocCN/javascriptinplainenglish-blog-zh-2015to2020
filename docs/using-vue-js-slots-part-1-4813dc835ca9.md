## 那么到底什么是老虎机？

插槽被认为是 Vue 组件的高级功能，但您将在任何生产级应用程序中使用它们。事实上，即使你使用了 [Bootstrap Vue](https://bootstrap-vue.org/) 库，你也需要理解插槽是如何工作的。
