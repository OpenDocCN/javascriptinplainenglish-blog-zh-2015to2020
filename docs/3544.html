object . getownpropertymanames()快速介绍

Object.getOwnPropertyNames()方法返回在给定对象中直接找到的所有属性(包括不可枚举的属性)的数组

语法

Object.getOwnPropertyNames(obj)

示例

Object.getOwnPropertyNames()返回对象的所有键

[ '0', '1', '2' ]
string

Object.keys()快速介绍

Object.keys()方法返回一个给定对象自己的可枚举属性名称的数组，迭代顺序与普通循环相同

语法

Object.keys(obj)

例子

Object.keys返回与Object.getOwnPropertyNames()相同的输出，因为obj中的每个元素都是可枚举的

[ '0', '1', '2' ]
string

最后，Object.getOwnPropertyNames()与Object.keys()

Object.getOwnPropertyNames(obj)返回对象的所有属性。Object.keys(obj)返回所有可枚举属性。除非您将enumerable: false设置为任何属性，否则它们会提供相同的结果

让我们再定义一个名为resolution的属性，但它被设置为enumerable: false

resolution属性则不会出现在Object.keys列表中

key: {enumerable: false | true,},

→所有属性(包括不可枚举的属性)
