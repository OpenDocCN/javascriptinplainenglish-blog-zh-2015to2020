“v-model”是 Vue.js 框架中的一个指令。正如官方 API 文档所解释的，v-模型是实现双向数据绑定的方法。这到底是什么意思？简单来说，就是当你在的文本区域写下一些东西或者从<select>中选择一个选项时，它会交互地显示你刚才在指定的显示区域写下或者选择的相同的值，你把{{ }}放入你的 JavaScript 文件中。以下是使用 CodePen 演示的基本用法。</select>
