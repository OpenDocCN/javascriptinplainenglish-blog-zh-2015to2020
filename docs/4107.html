如何修复GitHub上的“丢失Travis CI构建”

原文：https://javascript.plainenglish.io/fix-missing-travis-ci-build-on-github-3fcd1508411b?source=collection_archive---------12-----------------------#2020-11-16

背景

Travis CI 不再出现在我的 GitHub PR(拉取请求)中。我所做的是重新设置Travis CI应用程序对我的GitHub帐户的访问，以使构建再次显示出来。

撤销访问权限

打开 设置 :

然后进入 申请 > 授权OAuth申请:

从 Travis CI for Open Source 下拉列表中点击 Revoke :

重新授权申请

注销 Travis CI ，重新登录，并重新授权应用程序访问GitHub帐户:

打开一个新的PR后，我可以看到Travis CI构建在合并拉取请求按钮旁边。
