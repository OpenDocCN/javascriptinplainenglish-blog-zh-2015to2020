首先你必须安装一个名为ng-connection-service的软件包。让我们看看如何做到这一点。

使用CLI创建新的角度应用程序

ng new Online

使用以下命令从CLI在应用程序中安装ng-connection-service包:

npm install ng-connection-service — save
