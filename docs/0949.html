someFunction(str){

默认参数值

这是通常的实现方式

function area(h,w){
   if(!h){
     h=1;
   }  if(!w){
    w=1;
  }
  return h*w;
}

但是你可以用这个来代替。

function area(h=1,w=1){
  return h*w;
}
