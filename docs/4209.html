描述

给定一组会议时间intervals，其中intervals[i] = [starti, endi]，确定一个人是否可以参加所有会议。

例1: 

Input: intervals = [[0,30],[5,10],[15,20]]
Output: false

例2: 

Input: intervals = [[7,10],[2,4]]
Output: true

约束:

  • 0 <= intervals.length <= 104
  • intervals[i].length == 2
  • 0 <= starti < endi <= 106
