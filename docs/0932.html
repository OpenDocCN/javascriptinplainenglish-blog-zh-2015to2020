MobX (v5+)
MobX库已经证明了其作为状态管理库的核心，它促进了反应式编程方法，并且与redux相比，不太复杂，易于学习，样板代码非常少。

yarn add mobx

为react-native设置mobx对于第一次使用的人来说可能会很困惑，因为这个库依赖于使用decorator(es6特性)。

MobX-react (v5+)
为了让“mobx”与reactJs一起正常工作，我们还需要“Mobx-react”包，这个包充当“Mobx”库和“ReactJs”之间的链接，这让我们可以在react中使用Mobx的功能。

yarn add mobx-react
