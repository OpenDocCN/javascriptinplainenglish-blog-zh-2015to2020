## 让我们言归正传

1.  打开你的终端
2.  这会带你回家
3.  `touch .bash_profile`
4.  然后打开它输入`open .bash_profile`。如果你不在你的主目录中，你可以用这个命令`open ~/.bash_profile`打开它

每当您对您的`.bash_profile`进行更改时，确保运行这个命令来更新它

```
source ~/.bash_profile
```

希望这有帮助！
