让我们看看Angular CLI 7的6个最有用的命令

ng新【选项】:

  1. flag "—minimal= true | false "(默认为false) —创建新项目的最小版本，而不测试框架。
  2. flag“—路由=真|假”——是否生成路由模块。
  3. flag "-style= CSS | scss | sass | less "(默认为-scss)-样式文件的文件扩展名。

让我们在链中使用这些标志:

在创建当前组件时，也可以使用“ — style ”标志:

4.要查看在不创建文件的情况下将生成哪些文件，请使用标志“" dryRun =true|false "(默认为false) —非常有用的标志。您还可以在生成项目的步骤中使用它。

5.标志" — skipTests =true|false "(默认为false)用于跳过测试。

6.flag "—view encapsulation= Emulated | Native | None | shadow DOM "(默认情况下— Emulated) —视图封装策略。
