## 5.空与未定义

`null`是值，而`undefined`不是。`null`如同一个空的**盒子**，`undefined`根本就不是盒子。前任。,

![](img/78e0555ea10073e3d75c766dd4c12f16.png)

当`null`通过时，不取默认值，而当`undefined`或没有通过时，取默认值。

## 6.真实和虚假的价值观

**Falsy 值** — `false`、`0`、`""`(空字符串)、`null`、`undefined`、&、`NaN`。

**真值** — `"false"`、`"0"`、`{}`(空物)、&、`[]`(空阵)
