对于初学者，在 SvelteJS 资源库中有一个模板项目。您可以通过运行下面的命令来克隆此项目。

npx degit sveltejs/template template-master

在克隆了模板项目之后，您可以很容易地从 localhost:5000 端口发布项目。

cd template-masternpm install
npm run dev
